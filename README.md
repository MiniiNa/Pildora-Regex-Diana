## 🧠 ¿Qué es Regex?  

Regex (Regular Expressions) es un “lenguaje” compacto que permite:  

🔍 Buscar patrones en texto  
✅ Validar datos (emails, teléfonos, etc.)  
✂️ Extraer información relevante  
🔄 Reemplazar contenido automáticamente  

En esencia, es una secuencia de caracteres que define un patrón de búsqueda dentro de un texto.  

## 🧩 Conceptos básicos 
Símbolo	Significado  
.	Cualquier carácter  
*	0 o más repeticiones  
+	1 o más repeticiones  
?	0 o 1 repetición  
^	Inicio del texto  
$	Fin del texto  
[]	Conjunto de caracteres  
()	Grupo
